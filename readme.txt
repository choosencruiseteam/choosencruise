File Structure-

.git - (GITHUB management folder, DO NOT MODIFY THIS)

Libraries - (Hold referenced libraries, example: JQuery)

PHP - (Hold PHP files)
    -API: Hold PHP files that will be interfaced via a GET request
    -Controllers: Hold PHP Files that will implement classes and methods
                    used in API files.

View - (Hold HTML, CSS, and Javascript files)
    -[HTML Pagename folder]: Place all files relevent to the HTML page within the
                              folder
                --JS: Hold Javascript files for the parent page

    -Index.html: Index page for the website
    -styles.css: Style sheet for whole website
